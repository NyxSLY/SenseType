# Overlay UI Design Spec — V1 冰蓝科技毛玻璃

## 概览

屏幕底部浮动状态条，用于语音输入的视觉反馈。tkinter + PIL 实现。

## 尺寸（逻辑像素 @96dpi，运行时 × DPI 缩放）

| 属性 | 值 |
|------|-----|
| 宽度 | 290px |
| 高度 | 44px |
| 比例 | ~6.6:1 |
| 圆角 | 12px |
| 底部间距 | 80px |

## 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 背景底色（深） | `rgb(8, 12, 28)` | 午夜深蓝 |
| 背景底色（浅） | `rgb(18, 28, 56)` | 渐变顶部 |
| 外发光边框 | `rgba(0, 140, 220, 0.22)` | 半透明青 |
| 内边框线 | `rgba(0, 175, 255, 0.43)` | 亮青 |
| 顶部高光 | `rgb(150, 205, 255)` | 玻璃反射 |
| 录音指示 | `#00E5FF` | 亮青 |
| 录音外环暗态 | `#003848` | 暗青 |
| 音量柱低 | `#0A4A80` | 深蓝 |
| 音量柱高 | `#00D4FF` | 亮青 |
| 识别中文字 | `#7EB8FF` | 冰蓝 |
| 识别动画点 | `#00D4FF` | 亮青 |
| 结果文字 | `#E8F4FF` | 冷白 |

## 字体

| 用途 | 字体 | 大小 |
|------|------|------|
| REC 标签 | Candara | 8pt × DPI |
| 正文/结果 | Candara | 9pt × DPI |

## 背景渲染（PIL 3× 超采样）

1. 垂直渐变底色（上浅下深）
2. 噪点纹理层（30% 密度，白色微粒，alpha 5-35）→ 磨砂质感
3. 顶部高光弧（alpha 渐隐，模拟玻璃反射）
4. 圆角蒙版
5. 外发光边框（高斯模糊 radius=6）
6. 内边框细线
7. LANCZOS 缩回 1×

## 状态

### 录音中
- 左侧：脉冲光环（外环呼吸 暗青↔亮青）+ 内核常亮
- 标签：`REC`
- 右侧：音量时间线（deque 滚动，左=历史 右=最新）

### 识别中
- 居中文字 + 三个序列脉冲点（依次亮起）

### 结果
- 居中文字，2 秒后自动隐藏

## 交互
- 可鼠标拖动（WS_EX_NOACTIVATE 不抢焦点）
- 窗口透明度 0.93
